### Insall 

## Install dependencies
```bash
npx expo install react-native-web@~0.19.6 react-dom@18.2.0 @expo/webpack-config@^19.0.0
```

## Install project
```bash
npm install .
```

Install the `Expo Go` app on iphone. 
Scan the QR Code and run web.

### Run on web
```bash
npm run web
```
