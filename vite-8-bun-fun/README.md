### Stack

* React 19
* Vite 8
* Bun
* TanStack Query
* TanStack Table
* TanStack Form

### Features

* Tab 1 - Product CRUD (create, read, update, delete)
* Tab 2 - Search products by name or category
* Tab 3 - Top 3 products by rating
* Tab 4 - Product report with stats and category breakdown
* Tab 5 - Feedback on products with star ratings

### How to run

```bash
./run.sh
```
